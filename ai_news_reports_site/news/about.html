Automated AI News Intelligence

TLDR News is a fully automated daily news aggregation and reporting system. Every day, it fetches headlines from 5 major news sources, runs sentiment analysis, generates structured AI reports with key themes and future predictions, and publishes everything as a static site — zero manual intervention required.

Pipeline Architecture

The backend is a sequential agent chain built in Python. Each agent is a single-responsibility class with its own fallback behavior, ensuring the pipeline never crashes — even when RSS feeds go down or the AI API is unreachable.

Agent Details

RSS Reader Agent
Fetches and parses RSS feeds from 5 news sources with User-Agent spoofing for resilience. Falls back to the most recent archived report if a feed is unreachable.
Curator Agent
Deduplicates headlines via normalized text matching (lowercase, alphanumeric, first 120 chars) and trims each category to 12 items max.
Sentiment Agent
Runs VADER lexicon-based sentiment analysis on each headline, averages scores with a 0.6x dampening factor to prevent extreme swings, and classifies as Positive, Negative, Neutral, or Mixed.
Report Writer Agent
Calls OpenAI's Structured Outputs API with strict JSON schema validation to generate key themes, notable headlines with "why it matters" context, and confidence-rated future predictions. Includes automatic model fallback chain.
Executive Summary Agent
Generates a cross-category executive summary synthesizing the day's most important developments.
Future Outlook Agent
Produces time-horizon forecasts: 24-72 hours, 1-4 weeks, and a watch list of emerging stories to monitor.
Publisher Agent
Writes daily JSON reports, generates 40+ static HTML pages with Schema.org JSON-LD, creates RSS feeds and XML sitemaps, and maintains a 45-month report archive.

Graceful Degradation

The system is engineered to never fail silently or crash the deployment pipeline. Multiple fallback layers ensure reports are always generated and published.

RSS Feed Down
Automatically loads headlines from the 45-month backup archive for that category.
OpenAI API Unreachable
Falls back through a model chain (gpt-4o-mini → gpt-4o), then generates a deterministic stub report with "Low" confidence rating.
No API Key (Local Dev)
Pipeline runs in full deterministic mode — all agents execute, AI reports are replaced with structured stubs.
Report Generation Error
Exception is caught at the orchestration layer; a fallback report is substituted and the pipeline continues to the next category.

Frontend Features

The dashboard is built with vanilla JavaScript, HTML5, and CSS3 — no frameworks, no build tools, no npm. Everything ships as three files.

Sentiment Trend Visualization
SVG polyline charts showing 30-day sentiment history per category with marked extrema, zero-line reference, and delta indicators.
Real-time Search & Filter
Instant headline filtering with cursor position preservation across re-renders.
Dark / Light Mode
Full theme system using CSS custom properties, persisted in localStorage, applied before first paint to prevent flash.
Responsive Sidebar Layout
Desktop sidebar collapses to a slide-in drawer on mobile with overlay, touch-optimized tap targets, and auto-close on category selection.
SEO & Social Sharing
Dynamic meta descriptions, OpenGraph tags, Schema.org JSON-LD, canonical URLs, auto-generated RSS feeds, and XML sitemaps.
Progressive Loading
Skeleton loaders with shimmer animation, staggered card reveal, scroll progress bar, and cache-busted data fetching.
